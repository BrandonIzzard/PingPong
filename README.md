# Pong Game

Learned to make an old school pong game using HTML5 Canvas. 2 Player pong with a score board.

-----------------------------------------
![Site Pic](http://i.imgur.com/LqGGdiN.png)

#### Technology  & Tools Used

* HTML / HTML5 Canvas
* Webpack
* JavaScript ES5 & ES6
* CSS

##### Run

`> npm start`

## Keys

##### Player 1:
* a: up
* z: down

##### Player 2:
* ▲ : up
* ▼: down
